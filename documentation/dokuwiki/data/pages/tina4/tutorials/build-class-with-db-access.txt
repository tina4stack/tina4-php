====== Build a class with database access ======

Any class that extends **\Tina4\Data** will have a database variable **$this->DBA** to access the database from.

<code php>
<?php

class MyDbObject extends \Tina4\Data
{
    /**
    * Return all the fields in the test table
    **/
    function queryTheDB () {
        return $this->DBA->fetch('select * from table')->AsArray();
    }   
}
</code>

Or you can try this convention:

<code php>
<?php

class MyDbObject extends \Tina4\Data
{
    /**
    * Return all the fields in the test table
    **/
    function getSomethingFromDb() {
        return $this->DBA-->select ("*")->from("table")->asArray();
    }   
}
</code>
