====== Map my ORM field to a database field which doesn't follow the Tina4 pattern ======
If you require to extend or map your ORM object to a database field which doesn't follow the Tina4 pattern, take a look at the example below:

<code php>
<?php

public $fieldMapping = [
    "id" => "ID",
    "code" => "CODE",       
    "name" => "NAME"];
</code>
