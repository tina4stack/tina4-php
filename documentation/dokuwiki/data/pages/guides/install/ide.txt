======Install an IDE ======
You must have a code editor or tool which you will use to develop your website and use to debug. There are various IDE tools you can use to write code for your software or website,such as PHPStorm or Visual Studio Code. 

Depending on your Operating Software, you will need to install the correct installer or files for your OS. Follow these instructions to install an IDE tool on your system.

===== You may select one of these IDE tools =====
 * [[https://www.jetbrains.com/?from=Tina4| PHP Storm]]
\\
 * [[https://netbeans.apache.org/download/index.html| Netbeans ]] - requires Java installation & plugins]
\\
* [[https://code.visualstudio.com/download|Visual Studio Code]] - requires plugins for PHP to be added]]
\\
* [[https://atom.io|Atom]]
