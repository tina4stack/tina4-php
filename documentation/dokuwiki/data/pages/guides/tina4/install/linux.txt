====== How to Install Tina4 on Linux ======

===== Introduction =====
Follow the steps below to get Tina4 on your system. Soon you will be able to bring your ideas to life.

Make sure you have the prerequisites installed (php, composer, openssl & ide tool). 

===== Create a project =====
First you will need a folder in which Tina4 will be installed. In this example the folder is named tina4-php. Make your project directory by running this command in your terminal:

    mkdir your_project_folder
    
Once your project folder has been created, you will need to direct the command line to go into the folder you just created. In this example the folder is named "tina4-php". For more information on how to navigate directories with command line, please visit [[https://www.howtogeek.com/659411/how-to-change-directories-in-command-prompt-on-windows-10/|How To Geek]].
